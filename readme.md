## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/biblisearch.git
cd biblisearch
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to the provided local URL
